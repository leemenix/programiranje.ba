1. Napisati program koji ispisuje vase ime i prezime
```text

```

2. Napisati program koji crta pravougaonik oblika
```text
__________
|         |
|         |
|_________|
```

3. Napisati program koji crta trougao oblika
```text

  /    \
 /      \
/________\
```
